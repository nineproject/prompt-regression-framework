# TC-0010

Design an MVP for a user account-based bulletin board application.

## Requirements

- User registration
- Login / Logout
- Create post
- View post list
- View post detail
- Edit/Delete own posts only

## Expected Output

- Feature breakdown
- Screen structure
- Database schema proposal
- API design
- Authorization rules
- Implementation steps
- Security considerations

## Output Format

- Markdown
- Structured with headings
- Bullet-point oriented
- Easy to follow for implementation
